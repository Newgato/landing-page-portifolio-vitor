🚀 Landing Page Pessoal - Portfólio

Este projeto é uma landing page interativa desenvolvida com o objetivo de consolidar meus conhecimentos em **React** e servir como meu cartão de visitas digital. O design foi construído seguindo a tendência de **Glassmorphism**, focando em uma interface moderna, imersiva e totalmente responsiva.

🛠️ Tecnologias Utilizadas

* **React**: Biblioteca principal para a construção da interface modular.
* **CSS3**: Utilização de técnicas avançadas de estilização, incluindo filtros de desfoque (`backdrop-filter`) e variáveis para o tema neon.
* **Vite**: Ferramenta de build para um desenvolvimento rápido e otimizado.
* **FontAwesome**: Biblioteca de ícones para as seções de contato e redes sociais.

✨ Funcionalidades

* **Design Responsivo**: Adaptado para diferentes resoluções, desde desktops até dispositivos móveis e smart displays (como o Nest Hub).
* **Interface Glassmorphism**: Cartões com efeito de vidro fosco e bordas iluminadas.
* **Seções Dinâmicas**:
    * **Início**: Apresentação visual com partículas e efeitos neon.
    * **Sobre**: Resumo da minha trajetória como desenvolvedor e estudante de Ciência da Computação na **UniRitter**.
    * **Projetos**: Galeria com links para meus repositórios, incluindo meu **Bot de Assinaturas no Telegram**.
    * **Contato**: Atalhos diretos para WhatsApp e E-mail com tratamento de texto para telas pequenas.

🚀 Como rodar o projeto localmente

1.  Clone o repositório:
    ```bash
    git clone https://github.com/Newgato/landing-page-portifolio-vitor.git
    ```
2.  Entre na pasta do projeto:
    ```bash
    cd landing-page-portifolio-vitor
    ```
3.  Instale as dependências:
    ```bash
    npm install
    ```
4.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
